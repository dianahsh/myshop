
# Ecommerce Platform
"myshop" is a full-stack ecommerce web application that allows users to browse products, add items to cart, and place orders. 
--- 
## Features 
- User authentication (Signup/Login) 
- Product catalog with categories 
- Shopping cart 
- ...

--- 
## Tech Stack 
### Frontend 
- JS
- CSS 
### Backend 
- Django
### Database 
- postgresql
